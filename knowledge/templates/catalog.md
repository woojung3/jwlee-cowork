# Atlassian Consult Presentation Template Catalog

이 카탈로그는 각 슬라이드의 목적에 맞는 최적의 템플릿을 선택하기 위한 가이드입니다.

## 1. tpl-con-title
- **목적**: 발표의 시작을 알리는 메인 타이틀 슬라이드
- **구조**: 상단 이미지 영역 + 하단 제목/부제/날짜 영역
- **사용 시점**: 프리젠테이션의 가장 첫 페이지

## 2. tpl-con-splash
- **목적**: 대주제가 바뀔 때 사용하는 전환(Splash) 슬라이드
- **구조**: 중앙 집중형 텍스트, 강조하고 싶은 한 마디
- **사용 시점**: 새로운 챕터나 섹션이 시작될 때

## 3. tpl-con-default-slide
- **목적**: 일반적인 내용 전달용 슬라이드
- **구조**: 상단 타이틀 + 중앙 넓은 컨텐츠 영역
- **사용 시점**: 텍스트, 리스트, 또는 표를 넓게 보여줘야 할 때

## 4. tpl-con-default-box
- **목적**: 핵심 요약 또는 목표 강조 슬라이드
- **구조**: 상단 타이틀 + 라운드 박스로 감싸진 컨텐츠 영역
- **사용 시점**: 발표의 목표(Objectives)나 결론을 강조할 때

## 5. tpl-con-3-2
- **목적**: 비교 또는 상세 설명용 분할 슬라이드
- **구조**: 상단 타이틀 + 좌측 넓은 영역(3) + 우측 좁은 영역(2)
- **사용 시점**: 좌측에 개념 설명을 두고 우측에 핵심 요약을 배치할 때 (이미지+텍스트 조합 최적)

## 6. tpl-con-2-1-box
- **목적**: 복합 정보 배치용 슬라이드
- **구조**: 상단 타이틀 + 좌측 영역(2) + 우측 박스 영역(1)
- **사용 시점**: 좌측에 시각 자료(그림/차트)를 넣고 우측 박스에 짧은 설명을 넣을 때
